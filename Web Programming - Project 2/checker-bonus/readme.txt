Checker PW Tema2 cu Bonus, 2014
versiunea curenta: v0.1

============================================================
Atentie!
============================================================
Acest checker este valabil pentru variata cu bonus a temei.
Pentru varianta fara bonus este disponibil un alt checker in 
pagina temei.

============================================================
Configurare
============================================================
Configurati checker-ul in modul urmator:
- in fisierul settings.php inlocuiti valorile constantelor:
	STUDENT_ID - cu id-ul vostru de student pe care il 
	obtineti de la
	http://pw.dictionarit.eu/tema02-student-list.php 
	URL_TEMA2 - cu link-ul unde se afla tema voastra
- in fisierul js/checker.js inlocuiti valorile constantelor:
	STUDENT_ID - cu id-ul vostru de student pe care il 
	obtineti de la
	http://pw.dictionarit.eu/tema02-student-list.php 
	URL_TEMA2 - cu link-ul unde se afla tema voastra

============================================================
Rulare
============================================================
Rulati checker-ul in browser (mutati-l intr-un folder 
dedicat din /var/www sau echivalent) si dati click pe butonul
Check now
Checker-ul trebuie rulat DOAR pe tabelele curate, asa cum 
sunt extrase de pe 
http://pw.dictionarit.eu/tema02-student-list.php
